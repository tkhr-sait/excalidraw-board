## 貢献方法

### 課題報告
1. 新しい課題を発見した場合は、このディレクトリに新しいファイルを作成
2. ファイル名形式: `problem-XXX-短い説明.md`
3. テンプレートに従って課題を記録

### 課題解決
1. 各課題ファイルの「次のアクション」セクションを確認
2. 解決後は課題ファイルの「更新履歴」に記録
3. 関連するテストが合格することを確認
